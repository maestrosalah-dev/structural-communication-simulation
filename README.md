Structural Communication Simulation

This repository contains the agent-based simulation used in the paper:

"Modeling Structural Communication Dynamics: An Agent-Based Simulation of Endogenous Tension and Electoral Cascades"

The simulation models communication as structural transformation within adaptive networks. Symbolic flows interact with endogenous structural tension, producing opinion cascades and network polarization.

Main components:

- scale-free communication network
- endogenous structural tension
- symbolic flow influence
- relational rewiring
- cascade detection
